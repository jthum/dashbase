# Login Screen / Split

A page-level authentication shell that pairs contextual onboarding copy with the canonical login-box panel on larger viewports.

## Imports

```vue
<script setup lang="ts">
import { AuthLoginScreenSplit } from "@dashbase/vue/patterns/auth/login-screen/split";
</script>
```

```ts
import { AuthLoginScreenSplit, authLoginScreenSplitAssets } from "@dashbase/vue/patterns/auth/login-screen/split/manual";
```

## Scope

- Category: `auth`
- Family: `login-screen`
- Variant: `split`
- Scope: `page`

## Usage

```vue
<script setup lang="ts">
  import { AuthLoginScreenSplit } from "@dashbase/vue/patterns/auth/login-screen/split";
</script>

<template>
<AuthLoginScreenSplit />
</template>
```
