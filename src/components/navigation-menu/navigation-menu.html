Navigation Menu Component

Dashbase navigation menus style semantic <nav> directly when it contains anchored <popover-panel popover> children. The behavior shim only coordinates top-level roving focus and panel switching.

Use Left and Right to move across the root navigation.

When a panel is already open, moving across the root items will switch panels without leaving the keyboard. Unlike menubar menus, the panels can contain richer content because they are not constrained to strict role="menu" semantics.
