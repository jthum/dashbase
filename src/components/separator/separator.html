Separator Component

Dashbase styles semantic <hr> and presentational elements with role="separator". Use horizontal separators for thematic breaks and vertical separators inside compact toolbars or split layouts.

Horizontal Separators

Semantic <hr> is the right choice when the divider marks a thematic change in content.

For purely visual layout separation, use an element with role="separator".

A stronger separator is available when you need a clearer break between dense regions.
