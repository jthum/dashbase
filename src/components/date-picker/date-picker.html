Date Picker Component

Dashbase composes <date-picker> from a semantic text input, a small trigger button, and an anchored <popover-panel> containing <calendar-view>. The picker stays light by reusing the calendar’s month/year controls, range selection, and optional time inputs.

Single Date + Range Pickers

Single-date pickers close as soon as a date is chosen. Range and time pickers stay open so you can complete the full selection before pressing Done.

Uses the richer calendar with an optional time field.
