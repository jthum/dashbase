Popover Component

Dashbase styles Popover API surfaces on non-dialog elements. This is best for lightweight, non-blocking utility panels. Placement is viewport-based for now; anchored trigger placement can be revisited when anchor positioning is a safer baseline.

Quick filters

Temporary controls that should not block the rest of the page.

Keyboard shortcuts

Small reference content works well as a centered popover panel.

  • G then P opens the project switcher.
  • / focuses the command search.
  • Esc closes the current transient surface.
