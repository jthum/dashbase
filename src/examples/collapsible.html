Collapsible Component

Dashbase uses <details class="collapsible"> for compact hide-and-show sections. This is intentionally lighter than the richer accordion shell and works well for optional configuration or supporting context.

Advanced filters

Use collapsibles for optional detail that should stay in the document flow, not for destructive confirmation or required task guidance.

Webhook retry policy

Retries back off exponentially across five attempts. Failed deliveries remain visible in the events log for 30 days.

Bare Variant

Why use a collapsible instead of tabs?

Use a collapsible when the content is still part of the same narrative flow and should stay stacked vertically once opened.
